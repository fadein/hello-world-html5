CSS Attributes

Attribute Description Example
background-image Specifies a background image (or images) for a document element. The first image given is drawn on top, and subsequent (comma-separtated) images are drawn beneath. 
background-image:
url("https://mdn.mozillademos.org/files/11991/startransparent.gif"),
url("WhiteBox.png");
background-size How large should the background image be displayed? 
/* scale the image as much as possible while maintaining the original aspect ratio.
Any remaining empty areas are filled in with background-color */
background-size: cover;

/* scale the image as much as possible while maintaining the original aspect ratio.
Parts of the image are trimmed off so that no empty areas show th rough. */
background-size: contain;

/* width only, height is implicitly "auto" */
background-size: 75%;

/* width + height */
background-size: 20% 40%;
background-attachment Whether the background image should scroll with the page content, or whether it should stay put while the content scrolls by atop it. 

/* image stays put */
background-attachment: fixed;

/* image is fixed to its containing element, and scrolls with it's element's contents */
background-attachment: local;

/* image is attached to its elements border, does not scroll with the element's contents */
background-attachment: scroll;
background-color Specifies a color to be drawn behind a document element. Can be used as a fallback for an image which cannot be loaded. 
background-color: seagreen;
background-color: #2e8b57;
background-color: rgb(46, 139, 87);
background-color: hsl(146, 50%, 36%);
width Width of an element. This is the horizontal dimension within the padding space; the innermost core of the CSS box model. 
width: 50%;
width: 200px;
margin Shorthand Specify the width of the margin component of an element per the CSS box model. Set this to auto on the horizontal dimension to center an element within its container. 
/* all sides */
margin: 10px;

/* vertical + horizontal margins */
margin: 10px auto;

/* clockwise procession from top, right, bottom, left */
margin: 1px 2px 1em 2em;
font-size Specifies the size of text. May be specified in a variety of units, or with names such as x-small, medium, large, etc. 
font-size: xx-small;
font-size: larger;
font-size: 0.9em;
font-size: 14px;
font-size: 120%;
font-family Admits the specification of a prioritized list of font names (or classes of fonts) for the browser to try to render. If the browser cannot find a font within the 1st listed family, the next one in the list is tried until a match is found. It is therefore a good idea to end the list with a generic font family so that the browser can at least fall back on something that you picked. 
font-family: "DejaVu Sans Bold";
/* generic font families: good fallback values */
font-family: serif;
font-family: sans-serif;
font-family: monospace;
font-family: cursive;
font-family: fantasy;
font-weight Specify the boldness of a font 
font-weight: bold;

/* one step lighter than the prevailing weight */
font-weight: lighter; 

/* normal weight */
font-weight: 400;
display Specify the type of box in which to render an element (inline, block, etc.). Specifying none will hide an element. 
display: block;
display: none;
border Shorthand Specify the appearance of an element's border along all four sides. The values this property can take are a combination of some of the more specific properties discussed below. 
border: 1px solid black;
border-width Shorthand How wide the border should appear. How much space is reserved for the border component of the CSS box model. 
/* make all sides the same*/
border-width: 1px;

/* top+bottom = 1cm, left+right = 2cm */
border-width: 1cm 2cm;

/* clockwise procession from top, right, bottom, left */
border-width: 0 1px 0 2px;
border-style Shorthand How to draw the lines comprising the border 
border-style: solid;
border-style: dashed;
border-style: double;
border-style: dotted;
border-color Shorthand Which color to draw the lines comprising the border 
border-color: seagreen;
border-color: #000000;
border-collapse Whether to merge the borders of adjacent elements. 
border-collapse: collapse;
border-collapse: separate;
border-bottom Specify an element's border, but only along its bottom 
border-bottom: 1px solid black;
vertical-align Place an element within its container relative to the top, middle, or bottom of its cell. 
vertical-align: super;
vertical-align: top;
vertical-align: baseline;
vertical-align: middle;
vertical-align: bottom;
vertical-align: sub;
text-align
  1. Specify the horizontal alignment for a text element
  2. Specify your political alignment
 
text-align: left;
text-align: center;
text-align: right;

CSS Selectors

Selector Description Example
:hover Select elements over which the mouse cursor is hovering 
tr:hover { background-color: rgba(160, 160, 160, 0.8); }
:visited Select links which have been visited 
a:visited { color: rebeccapurple; }
:first-of-type Select the first element of a type from amongst its siblings 
td:first-of-type {
vertical-align: middle;
text-align: center;
}
:nth-child(an+b) Pseudo-class to match an element from amongst its siblings. 
tr:nth-child(2n+1); /* selects the odd-numbered rows of a table */
tr:nth-child(odd);  /* idem. */
tr:nth-child(3);    /* selects every third element */

Color specifications

Notation Description Example
Hexadecimal notation Hexadecimal representation of RGB or RGBA channels. Setting the alpha channel controls the transparency of the resulting color. 
color: #FF0033;
background-color: #aabbccdd;
RGB function notation Specify a color using decimal numerals within what appears to be a function call. Opaque colors may be specified with the rgb() functional notation. The alpha (transparency) channel can be specified as a percentage or as a fractional number between (0.0, 1.0) 
color: rgb(255, 0, 51);
background-color: rgba(170, 187, 204, 221);
HSL function notation With syntax congruent with the rgb() and rgba() notations, specify a color in the HSL(A) (hue, saturation, lightness (alpha)) color space. 
color: hsl(192, 60%, 51%);
background-color: hsla(170, 20%, 89%, 95%);
Color name Specify a color by name. Obviously, different people will disagree on exactly what "Pale Violet Red" looks like, and so some standard list of colors must exist for this to be a viable option. 
color: seagreen;
color: rebeccapurple;

Unix Commands

$ ls

funny-cat-gif-animation-1223.gif

cat
  1. Concatenate the contents of multiple files together to STDOUT
  2. Copy STDIN to STDOUT
 
$ cat < /dev/null 
These words go to /dev/null until I type EOF (ctrl-d)
^D
$
echo Print arguments to STDOUT 
$ echo one two 3 four
one two 3 four

$ stty

1486362244
